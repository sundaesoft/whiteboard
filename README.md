# 电子白板控件

## 产品概述

电子白板控件是一套基于C/S结构的成熟模块产品，适用于视频会议、教学等需要多人实时协作的场景。提供完整的源代码及开发文档，界面简单易用，功能丰富。

## 核心特性

### 基础功能
- **绘图工具**：直线、曲线、圆、椭圆、矩形、圆角矩形、橡皮擦、文字、标记、指示器
- **媒体支持**：插入图片、屏幕截图
- **文档操作**：打印文档、保存为白板文件

### 协作功能
- **多人协作**：支持多页白板，多用户实时协作
- **IE协同浏览**：多人IE协同浏览及批注
- **文档共享**：支持多人文档共享及批注
- **屏幕录制**：支持屏幕录制功能

### 文档格式支持
- **图片格式**：BMP、JPEG、GIF
- **文档格式**：Word、Excel、PowerPoint、PDF、TXT
- **Office支持**：包括Office 2007及以上版本

### 技术特性
- **C/S架构**：客户端/服务器架构，支持局域网和互联网
- **高性能**：服务器端采用完全端口模式，支持大规模并发
- **可扩展**：封装成DLL及OCX，调用方便，支持浏览器调用
- **多语言支持**：支持在VB6、Delphi7、VC2003及.NET环境中调用
